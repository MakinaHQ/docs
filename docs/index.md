# Introduction to Makina

Welcome to the Makina documentation. This guide will help you understand the core concepts, architecture, and usage of the Makina protocol.

## What is Makina?

Makina is an innovative protocol for superior onchain execution. It redefines how smart contract vaults operate by providing a highly agile infrastructure that is used to execute a wide variety of cross-chain strategies.

## Core Components

Makina consists of several key components:

- **Machine**: The core abstraction that manages deposits, redemptions, and fees
- **Caliber**: Handles cross-chain operations and accounting
- **Governance**: Manages permissions, risk parameters, and system updates
